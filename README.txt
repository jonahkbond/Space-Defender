-A project based off of the classic 2D Space Invaders arcade game, built using the Unity video game engine and scripted in C#. 

-This project was a fun way to get to know the Unity 3d game engine and a fun way to to practice writing scripts in C#.

-To Play the game: The Space Defender.exe file runs the game.

-The scripts are located in the Space-Defender\Assets\Scripts folder.
